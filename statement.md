## 1. 광고 여러 개를 한 번에 수정할 수 있도록

### 현재 상태

- 광고 수정 시 한 건씩 수정 버튼을 눌러 개별 수정해야 함.
- 여러 광고의 상태나 날짜를 동일하게 변경하고 싶을 때 반복 작업 필요.

### 변경 후

- 목록에서 체크박스로 원하는 광고를 여러 개 선택한 뒤, "일괄 수정" 버튼으로 한 번에 수정 가능.
- 수정할 항목만 골라서 적용 가능. (예: 상태만 변경, 종료일만 변경 등)
- 선택하지 않은 항목은 기존 값 그대로 유지.
- 수정 가능 항목: 상태, 키워드, 순위, 상품 링크, 시작일, 종료일

### 세부 작업 내역

| 작업 내용 | 설명 |
|---|---|
| 일괄 수정 화면 제작 | 여러 광고를 동시에 수정할 수 있는 팝업 화면 신규 제작 |
| 일괄 수정 처리 기능 | 선택한 광고들에 입력한 값을 한꺼번에 반영하는 서버 처리 제작 |
| 목록 화면에 버튼 추가 | "일괄 수정" 버튼 추가, 광고 선택 시에만 활성화되도록 처리 |
| 권한 처리 | 기존 수정 권한과 동일하게 적용 (마스터/대행사만 수정 가능) |

### 예상 비용

| 항목 | 금액 |
|---|---|
| 일괄 수정 화면 제작 | 300,000 ~ 400,000원 |
| 일괄 수정 서버 처리 | 150,000 ~ 200,000원 |
| 버튼 추가 및 연결 | 100,000 ~ 150,000원 |
| 테스트 및 검증 | 100,000 ~ 150,000원 |
| **소계** | **650,000 ~ 900,000원** |

## 2. 네이버 광고와 쿠팡 광고를 분리해서 관리할 수 있도록

### 현재 상태

- 네이버 광고와 쿠팡 광고가 한 페이지에 섞여서 표시됨.
- 플랫폼(네이버/쿠팡) 구분 없이 모든 광고가 같은 목록에 노출됨.

### 변경 후

- "네이버 광고" 페이지와 "쿠팡 광고" 페이지 완전 분리.
- 좌측 메뉴에서 "네이버 광고", "쿠팡 광고"를 각각 선택하여 이동 가능.
- 광고 등록 페이지도 플랫폼별로 분리.
  - 네이버 광고 등록 페이지에서 등록한 광고 → 네이버 광고로만 등록됨.
  - 쿠팡 광고 등록 페이지에서 등록한 광고 → 쿠팡 광고로만 등록됨.
- 각 페이지의 통계(전체/정상/오류/대기/종료예정/종료)도 해당 플랫폼 광고만 집계.
- 기존에 등록된 광고는 상품 링크를 기준으로 자동 분류.
  - 링크에 "naver"가 포함된 광고 → 네이버 광고
  - 그 외 → 쿠팡 광고

### 세부 작업 내역

| 작업 내용 | 설명 |
|---|---|
| 광고 데이터에 플랫폼 구분 추가 | 각 광고가 네이버인지 쿠팡인지 구분하는 정보를 저장할 수 있도록 함 |
| 기존 광고 자동 분류 | 이미 등록된 광고들을 상품 링크 기준으로 네이버/쿠팡으로 자동 분류 |
| 네이버 광고 목록 페이지 제작 | 네이버 광고만 표시되는 전용 목록 페이지 제작 |
| 쿠팡 광고 목록 페이지 제작 | 쿠팡 광고만 표시되는 전용 목록 페이지 제작 |
| 네이버 광고 등록 페이지 제작 | 네이버 전용 광고 등록 페이지 제작 |
| 쿠팡 광고 등록 페이지 제작 | 쿠팡 전용 광고 등록 페이지 제작 |
| 좌측 메뉴 변경 | 기존 "광고관리" 메뉴를 "네이버 광고" / "쿠팡 광고" 두 개로 분리 |
| 서버 처리 수정 | 목록 조회, 등록, 수정, 삭제 시 플랫폼별로 처리되도록 수정 |

### 예상 비용

| 항목 | 금액 |
|---|---|
| 데이터 구조 변경 및 기존 광고 분류 | 150,000 ~ 200,000원 |
| 네이버/쿠팡 목록 페이지 제작 | 500,000 ~ 700,000원 |
| 네이버/쿠팡 등록 페이지 제작 | 300,000 ~ 450,000원 |
| 좌측 메뉴 변경 | 100,000 ~ 200,000원 |
| 서버 처리 수정 | 300,000 ~ 450,000원 |
| 테스트 및 검증 | 200,000 ~ 350,000원 |
| **소계** | **1,550,000 ~ 2,350,000원** |

## 3. 네이버 광고의 키워드 순위를 자동으로 조회할 수 있도록

### 현재 상태

- 광고의 순위를 관리자가 직접 입력해야 함.
- 실제 네이버 쇼핑에서 몇 위에 노출되는지 확인하려면 직접 검색해서 찾아야 함.

### 변경 후

- 시스템이 하루 4회(0시, 6시, 12시, 18시) 자동으로 네이버 쇼핑을 검색하여 각 광고의 키워드 순위를 갱신함.
- 관리자가 별도로 버튼을 누르거나 조작할 필요 없이, 순위가 자동으로 최신 상태로 유지됨.
- 조회 결과는 광고의 순위 항목에 자동 반영.
- 100위권 안에 없는 경우 "-"으로 표시.
- 마지막으로 순위를 조회한 시각도 함께 표시.
- 네이버 광고 전용 기능. (쿠팡 광고 페이지에는 해당 없음.)

### 참고 사항

- 네이버에서 제공하는 공식 검색 서비스를 이용하여 순위를 가져옴.
- 검색 결과의 일반(자연) 노출 순위 기준이며, 광고 영역(AD 표시 영역)의 순위는 포함하지 않음.
- 네이버 서비스 이용을 위해 별도의 네이버 개발자 계정 등록 필요. (무료)
- 상품 링크가 네이버 스마트스토어 형식(`smartstore.naver.com/...`)이어야 순위 조회 가능.
- 키워드가 입력되지 않은 광고는 순위 조회 대상에서 제외.

### 세부 작업 내역

| 작업 내용 | 설명 |
|---|---|
| 네이버 검색 서비스 연동 | 네이버 쇼핑 검색 결과를 가져오는 기능 제작 |
| 상품 링크에서 상품 정보 추출 | 스마트스토어 링크에서 상품 번호를 자동으로 읽어내는 기능 제작 |
| 자동 순위 갱신 처리 | 하루 4회(6시간 간격) 모든 네이버 광고의 순위를 자동으로 조회·저장하는 예약 작업 제작 |
| 조회 시각 표시 | 순위 옆에 마지막으로 조회한 시각 표시 |
| 네이버 개발자 계정 설정 | 순위 조회에 필요한 네이버 계정 키를 시스템에 등록 |
| 예약 작업 환경 설정 | 자동 실행 스케줄(0시/6시/12시/18시) 등록 및 보안 설정 |

### 예상 비용

| 항목 | 금액 |
|---|---|
| 네이버 검색 서비스 연동 | 300,000 ~ 450,000원 |
| 상품 정보 추출 및 순위 매칭 | 300,000 ~ 450,000원 |
| 자동 순위 갱신 예약 작업 제작 | 400,000 ~ 600,000원 |
| 화면 작업 (조회 시각 표시) | 100,000 ~ 150,000원 |

### 비고

- 위 금액은 부가세 별도 기준.
- 요구사항 변경이나 추가 협의에 따라 금액이 달라질 수 있음.
- 기능 간 의존 관계가 있어 아래 순서로 진행.
  1. 먼저 네이버/쿠팡 분리 (기능 2) 완료 후
  2. 일괄 수정 (기능 1)과 순위 조회 (기능 3) 진행.
- 기능 3의 순위 조회는 네이버 광고에만 적용.
- 기능 3의 순위는 네이버 쇼핑의 일반 검색 결과 기준이며, 광고 영역(AD) 순위와는 다름.
- 네이버 검색 서비스 자체는 무료이며, 별도 이용료 없음.
