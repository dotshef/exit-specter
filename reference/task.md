# 공지사항 추가 작업
해당 SaaS에 공지사항 기능을 추가해야한다.

### 권한
오로지 총판(MASTER)만 등록, 수정, 삭제할 수 있다.
대행사(AGENCY)는 조회만 가능하다.
광고주(ADVERTISER)는 조회도 불가능하다.

### LNB
MASTER, AGENCY가 로그인했을 때에만 LNB 메뉴의 제일 위에 위치한다.

### 페이지
- 공지사항 목록 페이지
  - 하나의 row에서 제목과 작성 날짜, 조회수가 보인다.
  - 페이지네이션이 적용된다.
- 공지사항 상세 페이지
  - 최상단에 "목록으로" 버튼이 있다. 이 버튼을 누르면 공지사항 목록 페이지로 돌아간다.
  - 제목과 작성일자, 조회수가 위에 보인다.
  - 작성일자는 2025-12-12 와 같은 형태로 보이게 된다.
  - 본문이 아래에 보인다.
  

### 데이터 타입 (postgres)
- 테이블 명: notice
- 제목: text
- 내용: text
- 작성날짜: timestamp
- 조회수: int4
