# 광고 등록과 수정 메커니즘 변경

### 광고 수정 변경사항
- 광고 수정에서 수량은 제외한다.
- 광고 정보에서 수량 자체를 제외한다. 
- DB 테이블에서 수량 컬럼 제거 예정

### 광고 등록 변경사항
- 광고 등록은 모달이 아니라 별도의 페이지에서 실행한다.
- 광고 등록 페이지에서 초기 입력창에 내용을 입력한 후에 적용 버튼을 누르면 후기 입력창이 뜬다.
- 초기 입력창 입력 내용
  - 광고주 아이디 선택
  - 유형 선택 (광고 or 테스트)
  - 수량
- 후기 입력창
  - 수량 개수 만큼 입력 row 생성
  - 각 입력 row에서는 키워드, 상품링크, 시작일, 종료일 입력
  - 상태는 기본적으로 대기로 시작
  - 순위는 기본값 null
  - 남은 작업 일수는 현재날짜 - 종료일로 자동입력

초기 입력창은 위에 배치한다. 
초기 입력창이라고 명시하지 않는다. 등록페이지 UI에서 초기에 보일 뿐이다.
